## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## SEO Enhancements

### OpenGraph Image
To improve social media sharing, create an `og-image.png` file with the following specifications:
- Place it in the `/public` folder
- Size: 1200x630 pixels
- Content: Include your name, title, and possibly a visual representation of your skills/projects
- Format: PNG with transparent background recommended
- Make sure the text is readable when the image is shown at smaller sizes

This image will be used when your site is shared on social media platforms like Twitter, Facebook, and LinkedIn.

### Icons
The site now supports various icon formats for different platforms. Create the following icons and place them in the `/public` folder:

1. **Basic Icons**
   - `favicon.ico` - 16x16, 32x32, 48x48 (ICO format with multiple sizes)
   - `icon.svg` - Vector SVG icon
   - `icon-16x16.png` - 16x16 PNG icon
   - `icon-32x32.png` - 32x32 PNG icon

2. **Apple Touch Icons**
   - `apple-icon.png` - 180x180 PNG

3. **Android Icons**
   - `android-chrome-192x192.png` - 192x192 PNG
   - `android-chrome-512x512.png` - 512x512 PNG

4. **Other Platform Icons**
   - `safari-pinned-tab.svg` - SVG icon with single color (for Safari pinned tabs)

You can use online tools like [Favicon Generator](https://realfavicongenerator.net/) to create these icons from a single source image.
